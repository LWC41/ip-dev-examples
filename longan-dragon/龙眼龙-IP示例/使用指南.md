## 系统概述

### 什么是水果IP开发系统？

这是一个模块化、可复用的IP开发工作流系统，专为水果类IP设计而打造。它将IP开发过程分解为4个独立但相互关联的Skill，每个Skill负责特定领域的任务。

### 核心优势

✅ **模块化设计** - 4个独立Skill，可单独使用或组合使用
✅ **可复用性** - 一次设计，多次复用于不同水果IP
✅ **完整工作流** - 从角色设计到商业化，全流程覆盖
✅ **标准化输出** - 规范的文档格式和交付物
✅ **AI集成** - 集成Meshy API等AI工具，提高效率

### 适用人群

- IP创作者和设计师
- 农产品品牌方
- 文创公司
- 游戏开发商
- 跨界合作方

### 核心Skill

| Skill | 功能 | 适用阶段 |
|-------|------|---------|
| **ip-core-design** | 核心IP设计 | 早期 - 角色设计、世界观构建 |
| **ip-3d-generator** | 3D模型生成 | 中期 - 3D建模、场景搭建 |
| **ip-content-creator** | 内容生产 | 持续 - 表情包、插画、短视频 |
| **ip-commercialize** | 商业化运营 | 后期 - 周边产品、授权合作 |

## 详细使用说明

### 1. ip-core-design - 核心IP设计

#### 功能
- 角色设计（主角色、配角）
- 世界观构建
- 视觉系统规范

#### 使用步骤

**Step 1: 需求分析**
```
用户输入: "帮我设计一个广西龙眼的IP角色"
系统会询问：
- 目标受众？(儿童/青少年/成人)
- 风格偏好？(可爱/潮流/写实/Q版)
- 核心价值观？(健康/快乐/自然)
```

**Step 2: 角色设计**
系统会生成完整的角色档案，包括：
- 基本信息（名字、品种、性格）
- 外观特征（色彩、眼睛、配饰）
- 背景故事（出生地、梦想）

**Step 3: 世界观构建**
- 地理环境
- 社会规则
- 传说故事

**Step 4: 视觉系统**
- 色彩规范（Hex色值）
- 风格指南
- 表情包标准

**输出文档**
- `IP设计文档.md`
- `视觉规范.md`
- `角色档案表.md`

### 2. ip-3d-generator - 3D模型生成

#### 功能
- 文本转3D角色模型
- 文本转3D场景
- 批量生成角色家族

#### 前置配置

**设置API密钥**

```bash
# 方法1: 环境变量
export MESHY_API_KEY="your-api-key-here"

# 方法2: .env文件
在工作空间根目录创建 .env 文件
内容: MESHY_API_KEY=your-api-key-here
```

#### 使用步骤

**Step 1: 准备提示词**

根据ip-core-design输出的角色描述，构建提示词：

```python
prompt = """
A 3D character of Longan-Long, a cute anthropomorphic Guangxi longan character.

Appearance:
- Round and cute shape
- Golden brown skin (#8B6914)
- Big round eyes
- Small green leaf on top
- Wearing red scarf

Style:
- Cute cartoon style
- Soft lighting
- High quality 8k

Tags: cute, healing, child-friendly, high quality, 8k
"""
```

**Step 2: 调用API**

```python
from ip_3d_generator import LonganLong3DGenerator

generator = LonganLong3DGenerator()

# 生成3D模型
task_id = generator.create_text_to_3d(prompt)
result = generator.wait_for_completion(task_id)
```

**Step 3: 下载模型**

```python
# 下载GLB格式
model_url = result["model_urls"]["glb"]
generator.download_model(model_url, "longan-long.glb")
```

**Step 4: 批量生成角色家族**

```python
characters = [
    {
        "name": "Longan-Long",
        "description": "A cute anthropomorphic Guangxi longan character..."
    },
    {
        "name": "Litchi-Lily",
        "description": "A cute anthropomorphic litchi character..."
    }
]

results = generator.generate_character_family(characters, "3d-models/characters")
```

**输出文件**
- `3d-models/characters/[角色名].glb`
- `3d-models/scenes/[场景名].glb`
- `3d-generation-report.md`

## 常见问题

### Q1: Skill可以独立使用吗？

A: 可以！每个Skill都设计为可独立使用：
- 只需要角色设计 → 使用`ip-core-design`
- 只需要3D模型 → 使用`ip-3d-generator`
- 只需要表情包 → 使用`ip-content-creator`
- 只需要商业方案 → 使用`ip-commercialize`

### Q2: 如何获取Meshy API密钥？

A:
1. 访问 Meshy AI 官网: https://www.meshy.ai
2. 注册账号
3. 在设置中找到API密钥
4. 设置环境变量或创建`.env`文件

### Q3: 生成的3D模型质量不满意怎么办？

A:
1. 增加提示词的细节描述
2. 尝试不同的风格标签
3. 使用图像转3D功能，先生成2D参考图
4. 手动调整模型（使用Blender等软件）

### Q4: 表情包如何上传到微信？

A:
1. 确保有16个或以上表情
2. 表情尺寸统一（如512x512px）
3. 准备封面图（必须包含完整IP名称）
4. 访问微信表情开放平台: https://sticker.weixin.qq.com
5. 按要求提交审核

### Q5: 如何保护IP版权？

A:
1. 及时完成商标注册
2. 进行版权登记
3. 在所有素材上添加版权标识
4. 监控侵权行为
5. 签订正式授权合同

### Q6: 周边产品如何生产？

A:
1. 设计阶段: 使用ip-content-creator生成产品设计图
2. 打样: 联系供应商打样（MOQ通常500-1000件）
3. 质检: 检查样品质量
4. 生产: 批量生产
5. 销售: 通过电商、线下渠道销售

### Q7: 如何寻找品牌合作？

A:
1. 列出潜在合作品牌（食品、服装、文具等）
2. 准备授权方案和报价
3. 通过商务联系或展会接触
4. 提供合作方案和样品
5. 谈判并签订合同
