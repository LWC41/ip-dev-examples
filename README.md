# IP开发平台 - 示例项目

包含龙眼龙 IP 的完整示例代码、脚本和文档。

## 龙眼龙示例

广西龙眼 IP 的完整交付物，包括：

- IP 设计文档
- 3D 模型生成脚本
- 表情包生成脚本
- 短视频脚本库

### 使用示例

```bash
cd longan-dragon

# 设置 API Key
export MESHY_API_KEY=your_api_key

# 生成 3D 模型
python 3D模型生成脚本.py

# 生成表情包提示词
python 表情包生成脚本.py
```
